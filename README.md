### 🎓 Darabny — Enterprise Mentorship SaaS
**Software Engineering Graduation Project — FCAI, Assiut National University**

An enterprise-grade platform eliminating the *"Experience Paradox"* for fresh graduates — connecting students with corporate mentors through verified, real-world micro-internships. Built on strict **Clean N-Tier Architecture** with a polyglot backend combining **.NET 9** and **Python AI microservices**.

<img src="https://img.shields.io/badge/.NET_9-512BD4?style=flat-square&logo=dotnet&logoColor=white" />

| Highlight | Description |
|---|---|
| 🎙️ **AI Mock Interview** | Real-time conversational technical interviewer built with Python/FastAPI |
| 🧠 **AI CV Parsing & Matching** | `PdfPig` extraction + NLP scoring to rank mentors per student |
| ⚖️ **Double-Blind Evaluation** | Unbiased mutual ratings, unlocked only when both sides submit |
| 📜 **Cryptographic Certificates** | SHA-256 hashed, print-ready, publicly verifiable Experience Tokens |
| 🏛️ **Zero Data Loss** | Atomic EF Core transactions, soft deletes, strict cascade policies |
